GitHub

Ha a logóra kattintunk akkor eljutunk a GitHub oldalára.

Mi is az a GitHub? Ebben a videóban betekinthetünk a GitHub alapjaiba, a videó itt megtekinthető.

A GitHub egy online "raktár", egy repository amiben adatokat tudunk tárolni.
Ezen kívül a felületen egyszerre lehet dolgozni, ez alatt azt kell érteni hogy
ha valaki feltölti az elkészült munkáját akkor más azt át tudja írni ha van
engedéje ahhoz a repository megtekintéséhez.
