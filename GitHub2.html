Git első lépések

Magát a GitHub-ot erről a linkről tudjátok letölteni.

Érdemes a C vagy D meghajtóban dolgoznunk, így egyszerűbb lesz a dolgunk. Ha le is telepítettük a programot akkor nincs más mint létrehozni egy egyszerű repository-t egy egyszerű névvel.

Ezek után az azonosítás beállítása következik.(szükséghez a Github kapcsolathoz)

    a. git config --global user.name "Github-on megadott név"
    b. git config --global user.email "Github-on regisztrált emailcím"

A következő lépés hogy hozzunk egy lokális mappát a repositorynkban.

    a. git init

Következő, hozzunk létre egy fájlt a lokális mappában, adjuk hozzá
a fájlt az átmeneti tárolónkhoz.

    a. git add file-name

Minden aktuális mappában, almappában található fájl hozzáadása a staging-are-hoz.

    a. git add .

Piros: azon fájlok, amik nem kerültek be a staging-area-ba

Zöld: azon fájlok, amik berültek a staging-area-ba

státusz lekérdezés:

    a. git status

Átmeneti tárolóból a repoba való áttöltés

    a. git commit -m "Kötelező megjegyzést tenni"

Minden egyes bejegyzéshez a következőhöz hasonlót fogunk látni:

    commit a661d9249509f8aa8f81f6f58129d57eb36b81d6 Author: Gergo Roth
    Date: Wed Jul 6 18:36:01 2016 +0200
    My first commit.

Fájl és verzió feltöltése távoli tárhelyre

    a. git remote –v (kapcsolatot ellenőrizzük)
    b. git push origin master
